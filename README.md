# Sistema de Gestión de Torneos Deportivos 🏆
**Asignatura:** Programación 2  
**Institución:** Universidad Rafael Urdaneta (URU)  
**Trimestre:** 2026-C  

---

## 📝 Descripción del Proyecto
Este proyecto consiste en un sistema de consola en C++ diseñado para gestionar de forma completa e interactiva torneos deportivos, administrando equipos, jugadores y estadísticas de partidos. 

El desarrollo se enfoca en demostrar el dominio avanzado de **punteros, estructuras (`structs`) y la gestión manual de memoria dinámica**, siguiendo un principio estricto de **separación de capas** (Capa de Lógica de Negocio aislada de la Capa de Presentación)
